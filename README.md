# 🎮 마이크로비트 웹 조종기 (Micro:bit Web Controller)

## 📌 프로젝트 소개
이 프로젝트는 웹 브라우저의 **Web Bluetooth API**를 이용하여 인터넷 연결 없이도 마이크로비트(Micro:bit)를 원격으로 조종할 수 있는 **웹 기반 스마트 조종기 앱**입니다. 
학교, 학원 등 동일한 Wi-Fi 환경에서 수십 명의 학생이 동시에 접속하더라도 서버 차단이나 트래픽 문제가 전혀 발생하지 않도록 **1:1 로컬 블루투스 통신 방식**으로 설계되었습니다.

---

## ✨ 주요 기능

1. **간편한 무선 연결 (Web Bluetooth)**
   - 별도의 안드로이드/iOS 앱 설치 없이 웹 주소 접속만으로 블루투스 페어링이 가능합니다.
2. **직관적인 조이스틱 (가상 패드)**
   - 스마트폰 터치 화면에 최적화된 조이스틱을 통해 전, 후, 좌, 우 방향을 부드럽게 제어합니다. (P0, P1 핀 제어)
3. **4개의 커스텀 액션 버튼 (A, B, C, D)**
   - 로봇 팔, 집게(그리퍼) 등 추가 서보모터를 제어할 수 있는 4개의 다목적 버튼을 제공합니다. (P2 핀 제어)
4. **실시간 모터 튜닝 패널 (⚙️ 옵션 창)**
   - **가장 강력한 기능!** 마이크로비트의 MakeCode 블록을 매번 수정해서 다시 다운로드할 필요 없이, 앱 내부에서 모터의 속도와 각도를 즉시 미세 조정(Tuning)할 수 있습니다.
   - 직진이 안 되는 차량의 모터 오차를 잡거나, 집게의 벌어지는 각도를 조종기에서 바로 세팅할 수 있습니다.

---

## 🚀 사용 방법

### 1. 접속 및 블루투스 연결
1. 조종기 웹 앱 주소(배포된 웹사이트 링크)로 접속합니다.
2. 상단의 **[블루투스 아이콘]** 또는 **[연결하기]** 버튼을 누릅니다.
3. 팝업되는 기기 목록에서 자신의 마이크로비트(예: `BBC micro:bit [vevig]`)를 선택하고 연결합니다. 
   *(주의: 마이크로비트에 전원이 켜져 있고 블루투스 연결용 코드가 탑재되어 있어야 합니다.)*

### 2. 차량 조종 (조이스틱)
- 화면 중앙의 **조이스틱**을 손가락으로 밀어 차량을 움직입니다.
- 손을 떼면 즉시 조이스틱이 중앙으로 돌아오며 차량이 정지(STOP)합니다.

### 3. 추가 장치 조작 (액션 버튼)
- 화면 하단의 **A, B, C, D 버튼**을 누르면 마이크로비트의 P2 핀에 연결된 장치가 각각 설정된 각도로 즉각 움직입니다.

### 4. ⚙️ 모터 미세조정 (튜닝 패널 설정)
1. 화면 오른쪽 상단의 **⚙️ 톱니바퀴 아이콘**을 누르면 튜닝 설정 창이 열립니다.
2. **주행 모터 설정**: 왼쪽 모터(P0)와 오른쪽 모터(P1)의 전후좌우 수치를 수정합니다. 연속회전 서보모터의 경우 보통 90 근처가 정지, 0이나 180에 가까울수록 최고 속도입니다.
3. **버튼 모터 설정**: A, B, C, D 버튼을 눌렀을 때 P2 서보모터가 몇 도(0~180)로 회전할지 설정합니다.
4. 설정을 마치고 창을 닫으면 변경된 값이 저장되어 즉시 조종에 반영됩니다.

---

## ⚠️ 주의 사항 및 팁
- **지원 브라우저**: 안드로이드 기기 및 PC에서는 **크롬(Chrome)** 브라우저 사용을 권장합니다. (아이폰/아이패드의 경우 기본 사파리에서 블루투스 API를 지원하지 않으므로 앱스토어에서 `Bluefy` 라는 전용 웹 블루투스 브라우저를 다운로드하여 접속해야 합니다.)
- **수업 시 혼선 방지**: 교실 내에서 20명 이상이 동시에 연결을 시도할 때 서로 헷갈리지 않도록 마이크로비트 기기 뒷면에 견출지로 블루투스 고유 이름(예: `zizou`)을 꼭 붙여주세요.
