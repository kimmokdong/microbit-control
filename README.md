# 🎮 마이크로비트 웹 조종기 (Micro:bit Web Controller)
(만든이: 현승쌤)

## 📌 프로젝트 소개
이 프로젝트는 웹 브라우저의 **Web Bluetooth API**와 **File System Access API**를 이용하여 인터넷 연결 없이도 마이크로비트(Micro:bit)에 코드를 쉽게 넣고, 원격으로 조종할 수 있는 **웹 기반 스마트 조종기 앱**입니다. 
학교, 학원 등 동일한 환경에서 수십 명의 학생이 동시에 접속하더라도 쾌적하게 동작하도록 **1:1 로컬 블루투스 통신 방식**으로 설계되었습니다.

---

## ✨ 주요 기능

1. **원클릭 코드 설치 (직접 저장 방식)**
   - 번거로운 드래그 앤 드롭 없이, 웹사이트에서 `[코드설치]` 버튼을 누르면 마이크로비트 드라이브로 펌웨어 파일이 바로 복사됩니다.
2. **간편한 무선 연결 (Web Bluetooth)**
   - 별도의 안드로이드/iOS 앱 설치 없이 웹 주소 접속만으로 블루투스 페어링이 가능합니다.
3. **직관적인 방향 제어 및 액션 버튼**
   - 상/하/좌/우 방향 제어 및 4개의 커스텀 액션 버튼(A, B, C, D)을 통해 다목적 제어가 가능합니다.
4. **실시간 모터 튜닝 패널 (⚙️ 옵션 창)**
   - 마이크로비트 코드를 매번 수정할 필요 없이, 앱 내부에서 모터의 속도와 각도를 즉시 미세 조정(Tuning)할 수 있습니다.

---

## 🚀 사용 방법

### 1. 마이크로비트 코드 설치 (초기 1회)
1. 마이크로비트를 PC에 USB 케이블로 연결합니다.
2. 조종기 웹 앱에서 상단의 **[코드설치]** 버튼을 누릅니다.
3. '다른 이름으로 저장' 창이 뜨면, 왼쪽 목록에서 **`MICROBIT`** 드라이브를 선택하고 **[저장]**을 누릅니다.
4. 마이크로비트 뒷면의 노란 불빛이 깜빡이다 멈추면 설치가 완료된 것입니다!

### 2. 블루투스 연결 및 조종
1. 상단의 **[Connect]** 버튼을 누릅니다.
2. 팝업되는 기기 목록에서 자신의 마이크로비트(예: `BBC micro:bit [vevig]`)를 선택하고 연결합니다.
3. 화면의 방향키(상/하/좌/우)와 액션 버튼(A, B, C, D)을 눌러 마이크로비트를 조종합니다.

## 📱 기기별(OS별) 활용 가이드

교실 환경에 맞게 다음 가이드를 참고하여 수업을 설계하세요.

### 💻 1. 크롬북 (Chromebook) 및 윈도우(Windows) 노트북
**[가장 완벽한 베스트 환경]**
* **코드 설치**: 웹에서 `[코드설치]` 버튼을 누르면 뜨는 저장 창에서 `MICROBIT` 드라이브를 바로 선택할 수 있어 가장 편리합니다.
* **조종**: 크롬/엣지 브라우저를 통해 블루투스 조종이 완벽하게 지원됩니다.

### 📱 2. 안드로이드 태블릿 및 스마트폰 (갤럭시 등)
**[조종기로서의 활용도 최상]**
* **코드 설치 (비추천)**: 모바일 크롬은 보안상 '저장 창 띄우기'를 지원하지 않아, 파일이 기기의 다운로드 폴더에 받아집니다. 태블릿에 마이크로비트를 꽂고 파일 관리자 앱을 열어 수동으로 옮겨야 하므로 학생들에게는 매우 번거롭습니다.
* **조종 (추천)**: Web Bluetooth 기능을 완벽하게 지원합니다. 터치스크린 조작감이 좋으므로 리모컨 용도로 훌륭합니다.
* **💡 팁**: **코드는 PC/크롬북에서 미리 설치해두고, 태블릿은 조종기로만 활용**하는 것을 추천합니다.

### 🍎 3. 아이패드 (iPad) 및 아이폰
**[전용 브라우저 앱 필수 설치]**
* **조종 (기본 브라우저 불가)**: 애플의 보안 정책으로 인해 기본 Safari(사파리)나 iOS용 Chrome에서는 Web Bluetooth 기능이 차단되어 있습니다.
* **💡 팁**: 아이패드에서 조종기를 사용하려면 앱스토어에서 Web Bluetooth를 지원하는 전용 브라우저인 **`Bluefy`** 또는 **`WebBLE`**를 다운로드한 후, 해당 앱 안에서 웹사이트에 접속해야 합니다.

## 🚑 문제 해결 (Troubleshooting)

### Q. 블루투스 연결이 계속 실패하거나 튕깁니다!
가장 흔하게 발생하는 **'윈도우 블루투스 캐시(기억)' 오류**일 확률이 높습니다. 마이크로비트를 초기화하고 코드를 새로 넣었을 때 주로 발생합니다.
1. 윈도우 작업 표시줄 우측 하단 ⚙️ **[블루투스 및 기타 장치 설정]**을 엽니다.
2. 기존에 등록된 `BBC micro:bit`를 클릭하고 **[장치 제거]**를 누릅니다.
3. 윈도우의 **블루투스 스위치를 껐다가 3초 뒤에 다시 켭니다.**
4. 웹사이트를 강력 새로고침(Ctrl + Shift + R)한 후 다시 연결을 시도하세요.

*(참고: 만약 이래도 안 된다면, 원본 MakeCode 프로젝트 설정에서 `No Pairing Required`가 켜져 있는지 확인하세요.)*
