조종기 코드 설치 안내

파일이 성공적으로 다운로드되었습니다! 🎉

  1. 마이크로비트를 컴퓨터에 USB 케이블로 연결합니다.
  2. 방금 다운로드된 .hex 파일을 복사하거나 드래그합니다.
  3. 내 PC(내 컴퓨터)에 나타난 MICROBIT 드라이브 안으로 쏙! 붙여넣기 하세요.

파일 복사가 완료되면 마이크로비트 뒷면의 노란 불빛이 깜빡이다가 멈추고 재부팅됩니다.

연결 문제 해결 (도움말)

블루투스 연결이 계속 튕기거나 실패하나요?

마이크로비트를 초기화했거나 다른 코드를 넣었다면, 윈도우의 '블루투스 캐시(기억)'가 꼬였을 확률이 99%입니다.

  1. 윈도우 우측 하단 [블루투스 및 기타 장치 설정] 열기
  2. 목록에서 기존 BBC micro:bit 클릭 후 [장치 제거] 클릭
  3. 노트북의 블루투스를 완전히 껐다가 3초 뒤 다시 켜기
  4. 조종기 창을 새로고침(F5)하고 다시 Connect 시도!

그래도 안 된다면 원본 MakeCode 프로젝트 설정에서 'No Pairing Required'가 켜져 있는지, 코드 안에 '블루투스 UART 서비스' 시작 블록이 있는지 확인해 주세요.
